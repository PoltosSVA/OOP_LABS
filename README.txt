Class Fuel - класс,который хранит максимальное и текущее значение топлива
	Поля:
		current_fuel - текущее состояние топлива
		fuel_start - начальное кол-во топлива
		fuel_consumption - значение которое отнимается от стартового
	Методы:
		Start()-генерация первого объекта этого типа на карте
		-OnCollisionEnter2D(collision:Collision2D)- метод который проверяет столкнулись мы с обектом
		если да то он удаляется и обновляется стартовое значение топлива
Class Car_physics - класс, предназначенный для контроля физики текущей машины
	Поля:
		angle_speed - скорость поворота по 0y
		Rb - тело которое принимает данный скрипт
		wheelJoints - массив для хранения объектов тела
		back_wheel - тело заднего колеса
   		front_wheel -тело переднего колеса
		map - слой необходимый для обозначение карт
		br - слой для обозначения мостов
		back_wheel_Layer - объект принимающий значения для объектов колес и передающий им же
		ground - проверка на соприкосновение с землей
  		bridge - проверка на соприкосновение с мостом
 		max_speed - максимальная заданная скорость вперед
		max_back_speed - максимальная заданная скорость назад
		get_Stop - значение которое значительно сокращает скорость
		acceleration - ускорение
		deacceleration - торможение
 		break_force - сила сопротивления
		gravity - ускорение свободного падения
		axiesZ - значение для контроля поворота в воздухе
	Методы:
		Start() - получение значений стартовых обхектов
		Update() - корректировка стартовых значений
		FixedUpdate() - определяет физическую состовляющую игры
Class CameraControll - класс, который привязан к объекту и следит за ним при его перемещении
	Поля:
		dampTime - скорость следования за объектом
		velocity - необходим для следования в пространстве
		target - объект к которому привязан
	Методы:
		FixedUpdate() - покадровое перемещение камеры за объектом
Class Player - класс, который следит за стостояние текущего игрового объекта
	Поля:
		Rb - состояние тела обЪекта
		Fast_stop - экстренная остановка
		Stop - плавное торможение
		Gas - движение вперед
		Rotate_to_left - поворот против часовой стрелки
		Rotate_to_right - поворот по часовой стрелке
	Методы:
		waiter(...) - метод, который заканчивает игру спустя время после переворота объекта
		restart()  - перезапускает игру
		Update() - передача сигналов нажатых клавиш
Class Menu - класс,описывающий начальное игровое меню
	Поля:
		panel - бэкграунд
		sound - звук
	Методы:
		Pause() - пауза
		Continue() - продолжение
		Exit() - выход
Class StartTer класс, создающий стартовое пространство для движения объекта
	Поля:
		shape - объект играющий роль платформы
	Методы:
		Start() - процедурная генерация стартовой платформы
Class TertGer класс, отвечающий за процедурную генерацию карты и спавн объектов
	Поля:
		shape - объект играющий роль платформы
		fuel - объект топлива
		car - объект машины
		bridge - объект моста
		coin - объект монет
		pos - стартовая позиция объекта машины
		checker - проверка кол-ва использованых точек для процедурной генерации
		num_of_fuel_box - коэффициент насыщенности канистр топлива на карте
		num_of_coins - коэффициент насыщенности монетами на карте
		start_point - стартовая точка для процедурной генерации
		scale - высота холмов
		num_points - кол-во точек для процедурной генерации
	Методы:
		FixedUpdate()-покадровая проверка состояния карты и объектов
		AddFuel() - добавления объектов топлива на карту
		AddCoins() - добавление объектов монет на карту
		Start - инициализация полей
